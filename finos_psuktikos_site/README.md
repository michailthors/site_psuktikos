Site για επαγγελματικές ψυκτικές εγκαταστάσεις.

## Τεχνολογίες
- HTML/CSS
- Python/Flask

## Εκκίνηση
pip install -r requirements.txt
python app.py

## Δομή
templates/index.html → Το site
app.py → Flask server
